## 🚀 **Quick Start Guide**

*Get up and running in under 2 minutes*

</div>

### 📋 **Prerequisites**
```bash
✅ Node.js (v18 or higher)    📦 npm or yarn    🔑 TMDB API Key (Free)
```

<div align="center">

### 🎯 **Lightning-Fast Installation**

</div>

<table>
<tr>
<td width="50%">

#### **🔥 Method 1: One-Click Setup**
```bash
# Clone the magic
git clone https://github.com/sharmaram25/the-watchlist.git
cd the-watchlist

# Windows users
.\setup.bat

# Mac/Linux users  
chmod +x setup.sh && ./setup.sh
```

</td>
<td width="50%">

#### **⚡ Method 2: Manual Setup**
```bash
# Step 1: Clone repository
git clone https://github.com/sharmaram25/the-watchlist.git
cd the-watchlist

# Step 2: Install dependencies
cd client && npm install

# Step 3: Environment setup
echo "REACT_APP_TMDB_API_KEY=your_key_here" > .env

# Step 4: Launch the experience
npm start
```

</td>
</tr>
</table>

<div align="center">

### 🌐 **Launch The Experience**
Open your browser and navigate to `http://localhost:3000`
